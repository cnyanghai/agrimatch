# AgriMatch 字体使用规范

## 概述
项目统一使用 design-system.css 中定义的字体系统，确保全站字体样式一致。

## 字体大小系统

| 工具类 | 大小 | 用途 |
|--------|------|------|
| text-xs | 12px | 辅助文字、标签、徽章 |
| text-sm | 14px | 次要信息、描述文字 |
| text-base | 16px | 正文内容（默认） |
| text-lg | 18px | 小标题 |
| text-xl | 20px | 页面副标题 |
| text-2xl | 24px | 卡片标题、章节标题 |
| text-3xl | 30px | 页面主标题 |
| text-4xl | 36px | 首页主标题 |
| text-5xl | 48px | 营销标题 |
| text-6xl | 60px | 超大标题 |

## 字体粗细系统

| 工具类 | 粗细 | 用途 |
|--------|------|------|
| font-normal | 400 | 正文、描述 |
| font-medium | 500 | 强调文字 |
| font-semibold | 600 | 小标题 |
| font-bold | 700 | 主标题、按钮文字 |
| font-extrabold | 800 | 营销标题（限制使用） |
| ⛔ font-black | 900 | 禁止使用 |

## 行高系统

| 工具类 | 行高 | 用途 |
|--------|------|------|
| leading-none | 1.0 | 图标文字 |
| leading-tight | 1.25 | 标题 |
| leading-normal | 1.5 | 正文（默认） |
| leading-relaxed | 1.625 | 长篇文章 |
| leading-loose | 2.0 | 特殊排版 |

## 使用示例

### 页面标题
```vue
<div class="text-3xl font-bold leading-tight">控制台首页</div>
```

### 卡片标题
```vue
<h3 class="text-2xl font-semibold leading-tight">发布信息</h3>
```

### 正文内容
```vue
<p class="text-base font-normal leading-normal">
  这里是正文内容，使用默认的字体大小和行高。
</p>
```

### 辅助信息
```vue
<span class="text-sm text-gray-500 font-medium">最后更新：2024-01-01</span>
```

### 标签徽章
```vue
<span class="text-xs font-semibold px-2 py-1 rounded-full">新</span>
```

### 按钮文字
```vue
<button class="px-4 py-2 text-sm font-bold text-white">提交</button>
```

## 禁止事项

❌ **禁止使用 px 单位指定字体大小**
```vue
<!-- 错误 -->
<div style="font-size: 14px;">错误示例</div>

<!-- 正确 -->
<div class="text-sm">正确示例</div>
```

❌ **禁止使用数值直接指定字体粗细**
```vue
<!-- 错误 -->
<div style="font-weight: 800;">错误示例</div>

<!-- 正确 -->
<div class="font-bold">正确示例</div>
```

❌ **禁止使用 @apply 指令**
```css
/* 错误 */
.title {
  @apply text-2xl font-bold;
}

/* 正确 */
.title {
  font-size: var(--font-2xl);
  font-weight: var(--font-weight-bold);
}
```

❌ **禁止混用不同字体家族**
```vue
<!-- 错误 -->
<div style="font-family: Arial, sans-serif;">错误示例</div>

<!-- 正确 -->
<div class="font-sans">正确示例</div>
```

## 迁移指南

### 从内联样式迁移
```vue
<!-- 迁移前 -->
<div style="font-size: 20px; font-weight: 700; line-height: 1.25;">
  标题文字
</div>

<!-- 迁移后 -->
<div class="text-xl font-bold leading-tight">
  标题文字
</div>
```

### 从 Tailwind 迁移
项目使用自定义字体系统，与 Tailwind 兼容，但推荐使用设计系统工具类。

```vue
<!-- 两种方式都有效 -->
<div class="text-xl font-bold">标题</div>

<!-- 或者使用 CSS 变量 -->
<div style="font-size: var(--font-xl); font-weight: var(--font-weight-bold);">
  标题
</div>
```

## 响应式字体

如需响应式字体，可结合 Tailwind 的响应式前缀：

```vue
<div class="text-base sm:text-lg lg:text-xl">
  响应式标题
</div>
```

## 常见组合

| 用途 | 组合 |
|------|------|
| 页面主标题 | text-3xl font-bold leading-tight |
| 卡片标题 | text-2xl font-semibold leading-tight |
| 章节标题 | text-xl font-semibold leading-tight |
| 小标题 | text-lg font-semibold leading-snug |
| 正文 | text-base font-normal leading-normal |
| 辅助文字 | text-sm font-medium leading-normal |
| 标签 | text-xs font-semibold leading-none |
| 按钮 | text-sm font-bold leading-none |

## 注意事项

1. **一致性**：全站统一使用此规范，避免随意修改
2. **可读性**：正文最小字号为 text-sm (14px)，text-xs 仅用于标签和徽章
3. **可访问性**：保持适当的对比度和行高，确保可读性
4. **性能**：使用工具类而非内联样式，提高性能
