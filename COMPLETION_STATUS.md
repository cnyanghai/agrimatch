# 已完成 / 未完成清单（以代码与页面入口为准）

> 口径说明：
> - **已完成**：前后端与数据库（如需要）均已落地，能走通主流程。
> - **部分完成**：有入口/有部分实现，但存在明显占位、权限缺口、或依赖未齐。
> - **未完成**：仅规划/占位，或缺少核心后端/前端实现。

## 已完成（可用）

### 基础能力
- 统一返回 `Result`、全局异常处理
- Spring Security + JWT 无状态鉴权
- CORS（允许本地前端 5173）
- 文件上传与 `/uploads/**` 静态访问
- 数据库初始化脚本（核心表 + 品类数据）

### 账号与资料
- 注册 / 登录 / 获取当前用户（JWT）
- 我的档案：查看/更新，角色切换（采购/供应）
- 公司档案：创建/更新、营业执照图片上传与预览

### 业务模块
- 品类：树/搜索/自定义
- 采购需求：发布/列表/详情/更新/删除（逻辑删除），支持过期不展示
- 需求模板：保存/我的模板/删除
- 供应发布：发布/列表/详情/更新/删除（逻辑删除），带品类参数与自定义选项能力

### 社区/交易增值（已接入）
- 观点帖子：发布/列表/详情/删除 + 社交互动（以页面与接口为准）
- 积分：余额、充值/兑换、明细
- 通知：我的通知、已读/全部已读
- 即时聊天：HTTP 会话/历史/已读 + WebSocket `/ws/chat`

## 部分完成（可运行但有明显缺口/占位）

### 排序能力（距离/到厂价）
- 需求列表与供应列表里存在 `distance` / `delivered_price` 等排序项，但目前仍是占位或回退排序
- 尚未接入高德距离/到厂价计算链路

### 地图模块
- 前端存在 Map 相关页面/规划，但缺少完整联动（选点回填、聚合、导航等）

## 未完成（规划/占位为主）
- 智能匹配：按到厂价/距离/时间综合排序与推荐
- 业务闭环：成交/订单/合同模板落地、评价体系落地（若数据库已预置则仍需接口与页面）
- 更完善的权限模型：管理员/企业员工/数据隔离策略
